Mikrokontroler:     ESP32
Port szeregowy:     HardwareSerial (Serial1)

Połączenie:         TX → RX matrycy, GND wspólne

                            JAK ZACZĄĆ
═══════════════════════════════════════════════════════════════════════════════

1. Wybierz metodę instalacji:
   - Opcja A: Skopiuj folder LEDMatrix do Arduino/libraries/
   - Opcja B: Użyj pliku LEDMatrix_Standalone.ino

2. Podłącz sprzęt:
   - ESP32 TX (GPIO 17) → RX matrycy
   - ESP32 GND → GND matrycy

3. Otwórz przykład:
   - File → Examples → LEDMatrix → TestLEDMatrix
   - Lub otwórz LEDMatrix_Standalone.ino

4. Wgraj na ESP32 i ciesz się! 🚀
